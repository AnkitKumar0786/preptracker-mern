## ⭐ Benefits of using the above Tech Stack :
1. **MongoDB Atlas** as a database offers flexible NoSQL data models that easily handle complex user profiles and dynamic note structures.
2. **Node.js & Express** provide a blazing-fast, asynchronous backend environment perfect for handling multiple API requests simultaneously.
3. **React & Vite** deliver an incredibly fast, dynamic Single Page Application (SPA) experience with instant hot-reloading during development.
4. **Tailwind CSS** allows for rapid UI development with utility classes, ensuring the application is fully responsive across all mobile and desktop devices.
5. **Google Gemini AI API** integration powers the intelligent automated generation of structured Data Structures and Algorithms (DSA) study notes.
6. **JSON Web Tokens (JWT) & Bcrypt** ensure robust, industry-standard security through secure password hashing and stateless authentication via HTTP-only cookies.

---
## 🔥 Features
1. **Secure Authentication:** Complete Login and Registration system with encrypted passwords.
2. **AI Notes Generator:** Automatically generate structured DSA notes by simply typing a topic name, powered by Google Gemini.
3. **Goal Dashboard:** Set, edit, and track interactive target milestones (e.g., LeetCode problems solved, Codeforces ratings) with visual progress bars.
4. **Notes Management:** Full CRUD operations to create, read, update, and delete personalized study materials with Markdown support.
5. **Responsive Design:** A fully mobile-friendly experience featuring a dynamic hamburger navigation menu and clean UI.

## ✏️ Installation and Setup

**1. Clone the Repository:**
```bash
git clone [https://github.com/AnkitKumar0786/preptracker-mern.git](https://github.com/AnkitKumar0786/preptracker-mern.git)
cd preptracker-mern
```

**2. Setup the Backend (Server):**
```bash
cd server
npm install
```

**3. Configure Backend Environment:**
Create a `.env` file in the `server` directory and add your credentials:
```env
PORT=3000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_super_secret_key
GEMINI_API_KEY=your_google_gemini_api_key
```

**4. Start the Backend Server:**
```bash
npm run dev
```

**5. Setup the Frontend (Client):**
Open a new terminal window and navigate to the `client` folder.
```bash
cd client
npm install
```

**6. Configure Frontend Environment:**
Create a `.env` file in the `client` directory:
```env
VITE_API_URL=http://localhost:3000/api
```

**7. Start the Frontend Application:**
```bash
npm run dev
```
